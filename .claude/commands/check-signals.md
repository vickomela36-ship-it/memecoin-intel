# Check Memecoin Buy Signals

Run the memecoin signal checker and send alerts for any "buy now" signals.

## Step 1 — Run the signal script

```bash
cd /home/user/memecoin-intel && python3 signals.py
```

Parse the JSON output. If the top-level `"error"` key is present, report it and stop.

## Step 2 — Process each signal in `buy_signals`

For **every item** in the `buy_signals` array do the following two things:

### 2a — Log to Notion

Add a new row to the Notion data source `73b5d85d-86bf-4b6e-b8d7-c43e92bc0391` with these fields:

| Field | Value |
|---|---|
| Signal (title) | `BUY NOW - $<token_symbol>` |
| Coin | `$<token_symbol> / <token_name> on <chain>` |
| Price | `<price_usd>` (number) |
| Confidence | `<confidence>` (0–1 float) |
| Status | `New` |
| Email Sent | false (will update after email) |
| Notes | `24h: +<price_change_24h>% \| 6h: +<price_change_6h>% \| 1h: +<price_change_1h>% \| Vol24h: $<volume_24h> \| Liq: $<liquidity_usd> \| DEX: <dex> \| <pair_url>` |
| date:Timestamp:start | `<timestamp>` (ISO-8601) |
| date:Timestamp:is_datetime | 1 |

### 2b — Create a Gmail draft (email alert)

Create a Gmail draft to `vickomela36@gmail.com` with:

**Subject:** `🚨 Memecoin BUY Signal: $<token_symbol> (+<price_change_24h>% 24h)`

**HTML body:**
```
<h2>🚨 Memecoin Buy Signal Detected</h2>
<table>
  <tr><td><b>Token</b></td><td>$SYMBOL / Full Name</td></tr>
  <tr><td><b>Chain</b></td><td>chain</td></tr>
  <tr><td><b>Price</b></td><td>$price_usd</td></tr>
  <tr><td><b>24h Change</b></td><td>+price_change_24h%</td></tr>
  <tr><td><b>6h Change</b></td><td>+price_change_6h%</td></tr>
  <tr><td><b>1h Change</b></td><td>+price_change_1h%</td></tr>
  <tr><td><b>24h Volume</b></td><td>$volume_24h</td></tr>
  <tr><td><b>Liquidity</b></td><td>$liquidity_usd</td></tr>
  <tr><td><b>Confidence Score</b></td><td>confidence (0–1)</td></tr>
  <tr><td><b>DEX</b></td><td>dex</td></tr>
</table>
<p><a href="pair_url">View on DexScreener →</a></p>
<p><a href="https://app.notion.com/p/9ef4b2232d1a41649c867a47f8b4350f">View Signals Log in Notion →</a></p>
<hr><small>Generated by memecoin-intel at timestamp</small>
```

After the draft is created, update the Notion row's **Email Sent** field to `true` (checkbox = `__YES__`).

## Step 3 — Report results

Print a concise summary:
- How many tokens were checked
- How many buy signals were found
- For each signal: token symbol, confidence, Notion logged ✓/✗, email drafted ✓/✗

If no buy signals were found, print:
```
✓ Signal check complete — no buy signals detected at <checked_at>
  (Tokens scanned: <tokens_checked>)
```
